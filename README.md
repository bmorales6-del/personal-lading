## Cómo subirlo a GitHub Pages

1. Crea un repositorio nuevo en GitHub (por ejemplo: `portfolio-bayron`).
2. Sube **todo el contenido** de esta carpeta a la raíz del repositorio (no subas la carpeta comprimida, descomprímela primero).
3. Ve a **Settings → Pages** del repositorio.
4. En "Source", selecciona la rama `main` (o `master`) y la carpeta `/ (root)`.
5. Guarda. En unos minutos tu sitio estará disponible en:
   `https://tu-usuario.github.io/portfolio-bayron/`

## Notas

- El sitio es completamente responsivo (móvil, tablet, escritorio).
- Incluye un chatbot flotante con forma de rayo con respuestas automáticas.
- Los videos se cargan desde YouTube mediante iframes.
- No requiere ningún backend ni instalación: es HTML/CSS/JS puro.
